# TaskFlow (MongoDB + React/Tailwind) — Projeto pronto para portfólio

Versão profissional do TaskFlow usando MongoDB Atlas (Mongoose) no backend e frontend moderno com React + TailwindCSS.

## Instruções rápidas

1. Backend
- Copy `backend/.env.example` to `backend/.env` and set `MONGO_URI` and `JWT_SECRET`.
- From /backend: `npm install` then `npm run dev`.

2. Frontend
- From /frontend: `npm install` then `npm run dev`.

## Deploy
- Backend: Render / Railway (set environment variables `MONGO_URI` and `JWT_SECRET`).
- Frontend: Vercel (build command `npm run build`, output `dist`).
